📖This / That / These / Those

Esta é a regra mais importante da prova! Mostra se o animal está perto ou longe, e se é 1 ou vários.

📍 PERTO
🔭 LONGE
This is a cow.
(Esta é uma vaca)
That's a cow.
(Aquela é uma vaca)
These are goats.
(Estas são cabras)
Those are goats.
(Aquelas são cabras)

🔊 Ouça as 4 frases-modelo:

This is a duck.
That's a cow.
These are goats.
Those are chickens.
🎯
TRUQUE: Se o animal é 1 só → "This/That". Se são vários → "These/Those". Se está aqui pertinho → "This/These". Se está longe → "That/Those".

❓ Perguntas e Respostas

What's this? → It's a caterpillar.
What's that? → It's an ant.
What are these? → They're grasshoppers.
What are those? → They're mosquitoes.
